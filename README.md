# Placement Portal

A full-stack Placement Portal application built using Java, Spring Boot, React, and MySQL.

## 🚀 Tech Stack

- Java 17
- Spring Boot
- Maven
- REST API

## 📋 Features (Planned)

- Student Login
- Recruiter Login
- Admin Login
- Job Posting
- Apply for Jobs
- Resume Upload
- JWT Authentication
- MySQL Database
- React Frontend
